## Quick Start

### Prerequisites

- [Node.js](https://nodejs.org/) >= 22
- [Bun](https://bun.sh/) >= 1.2
- [PostgreSQL](https://www.postgresql.org/) >= 14

### Installation

```bash
# Clone the repository
git clone https://github.com/raypx/raypx.git
cd raypx

# Install dependencies
bun install

# Copy environment variables
cp .env.example .env

# Update .env with your configuration
```

### Database Setup

```bash
# Generate database client
bun run db:generate

# Push migrations
bun run db:push

# (Optional) Seed database
bun run db:seed
```

### Development

```bash
# Start development server
bun run dev

# Start specific app
bun run dev:web      # Web application (http://localhost:3000)
bun run dev:docs     # Documentation (http://localhost:3004)
```

## Performance Optimizations

Raypx includes several performance optimizations out of the box:

- **CSS Preloading** - Critical CSS is preloaded for faster initial render
- **Tailwind Minification** - Production builds include minified CSS
- **Optimized Builds** - Turbo-powered monorepo for fast builds
- **Code Splitting** - Automatic code splitting with TanStack Router
