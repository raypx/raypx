Welcome to the **Raypx** documentation. This guide will help you get started with our platform.

## Features

- **Modern Stack**: Built with Next.js 16 and React 19
- **Type-Safe**: Full TypeScript support
- **Internationalization**: Multi-language support out of the box
- **Beautiful UI**: Shadcn/ui components

## Quick Start

### Step 1

Install the dependencies:

```bash
pnpm install
```

### Step 2

Configure your environment variables:

```bash
cp .env.example .env
```

### Step 3

Start the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
