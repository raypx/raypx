## 介绍

Fumadocs <span className='text-fd-muted-foreground text-sm'>(发音：Foo-ma docs)</span> 是一个基于 Next.js 的**文档框架**，具备速度快、可定制性强、并且可以无缝兼容 App Router 等特性。

本测试文档用于验证多语言文档加载逻辑，涵盖以下内容：

- Fumadocs 的核心特点
- 项目内如何启用中文文档
- 接下来的学习资源

## 核心特点

1. **即开即用**：默认配置开箱即用，适合快速搭建文档站点。
2. **灵活架构**：支持根据业务需求组合组件，保持高扩展性。
3. **国际化友好**：内建多语言支持，便于为全球用户提供内容。

## 如何启用中文页面

1. 在 `apps/web/content/docs/` 目录下为文档添加 `*.zh.mdx` 文件。
2. 确保路由配置或文档框架正确识别语言后缀。
3. 重新构建或运行开发服务器以加载最新内容。
